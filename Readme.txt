Dataset link: https://data.gov.uk/dataset/road-traffic-accidents
(Data set is already in dataset folder)

Please install python libraries : pnadas, numpy, os, matplotlib.pyplot, kmodes.kmodes, itertools, glob 


In anaconda prompt change directory to Project.

(1) First run the data file using -- python data.py -- (it will combined all the dataset in one csv and create new 
combined.csv file.)

(2) Run data cleaning file using -- python data_cleaning.py -- (it will clean the data of combined.csv)

(3) Run the cluster.py file using -- python cluster.py -- (it will create clusters and save different files 
for all clusters in clusters folder)

(4) Run the As_Rules.py file using -- python As_Rules.py --  (You can change file name in python file to analysis 
other file)

(5) Run the features_analysis.py file using -- features_analysis.py --  (You can change file name in python file to analysis 
other file)

(6) Run the plot.py file using -- plot.py --  (You can change file name in python file to analysis other file)
